Stargazers


Stargazers is a interactive VR experience I helped to develop during my ARVR 300 class at SCAD, and while the project was worked on past then, I left the project to persue other endevors and do deal with occurences in my personal life. This project was developed in Unreal Engine 4. The project is being published on steam. Unlike the other projects I've worked on, this project was light on coding and we had four programmers on the project so I don't have anything specific to break down. This project was fun to work on and a good learning experience for VR game development.
